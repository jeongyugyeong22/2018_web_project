###강아지 물품 쇼핑 웹 사이트

##:open_book: Introduction

2018년 학교 프로젝트로 웹 사이트 제작 프로젝트를 진행하였습니다.
여러 강아지 물품을 판매할 수 있는 쇼핑몰로 컨셉을 잡아 진행하였습니다.
admin으로 로그인 시 관리자페이지에서 전반적으로 물품 관리, 회원 관리, 게시판 관리를 할 수 있도록 만들었습니다.
